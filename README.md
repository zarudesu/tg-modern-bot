# IT-Аутсорс: Централизованная система управления

## Описание проекта
Централизованная система для управления IT-аутсорсингом с поддержкой:
- Документирования сетей и устройств (NetBox)
- Безопасного хранения доступов (Vaultwarden)
- Базы знаний (BookStack)
- Telegram-бота для быстрого поиска
- Системы обращений клиентов

## Архитектура
- **Этап 1**: NetBox + Vaultwarden + BookStack + Telegram-бот
- **Этап 2**: Система обращений + LLM интеграция
- **Этап 3**: Миграция на Siyuan + умный бот + интеграция с Контур.Эльба

## Структура проекта
```
it-outsource-system/
├── docker/              # Docker конфигурации для всех сервисов
├── telegram-bot/        # Код Telegram-бота
├── custom-services/     # Собственные разработки
├── configs/             # Конфигурационные файлы
├── scripts/             # Скрипты деплоя и автоматизации
├── docs/                # Документация проекта
└── backup/              # Скрипты резервного копирования
```

## Клиенты и инфраструктура
- **Размер команды**: 3 разработчика
- **Клиенты**: 5-6 юридических лиц
- **Инфраструктура**: ~300 компьютеров, 20-30 серверов
- **Отрасли**: 3D-печать, медицина, строительство, базы отдыха

## Развертывание
Система разрабатывается локально с синхронизацией на продакшн-сервер через Git.
