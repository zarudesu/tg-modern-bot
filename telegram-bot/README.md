# 🤖 Modern Telegram Bot

Современный Telegram бот с модульной архитектурой, системой ролей и интеграциями.

## ✨ Особенности

- 🏗️ **Модульная архитектура** - легко добавлять и убирать функциональность
- 🔐 **Система ролей** - admin/user/guest с middleware авторизации
- 📊 **Полное логирование** - все действия записываются в БД
- 🐳 **Docker-ready** - готов к деплою в продакшн
- ⚡ **Async/await** - полностью асинхронный код
- 🔒 **Type hints** - полная типизация с Pydantic

## 🚀 Быстрый старт

### 1. Клонирование проекта
```bash
git clone <your-repo-url>
cd telegram-bot
```

### 2. Настройка окружения
```bash
# Копируем пример конфигурации
cp .env.example .env

# Редактируем .env файл с вашими настройками
nano .env
```

### 3. Запуск с Docker
```bash
docker-compose up -d
```

### 4. Или запуск для разработки
```bash
# Создаем виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем бота
python -m app.main
```

## ⚙️ Конфигурация

Основные переменные в `.env`:

| Переменная | Описание | Обязательная |
|------------|----------|--------------|
| `TELEGRAM_TOKEN` | Токен бота от @BotFather | ✅ |
| `ADMIN_USER_ID` | Telegram ID администратора | ✅ |
| `DATABASE_URL` | Строка подключения к PostgreSQL | ✅ |
| `REDIS_URL` | Строка подключения к Redis | ❌ |

## 🤖 Команды бота

### Основные команды
- `/start` - Регистрация и приветствие
- `/help` - Справка по командам
- `/profile` - Информация о профиле
- `/ping` - Проверка работоспособности

### Административные команды
- `/users` - Список пользователей (только админ)
- `/logs` - Просмотр логов (только админ)
- `/stats` - Статистика использования (только админ)

## 🔌 Модули

Бот построен по модульному принципу. Каждый модуль можно легко:
- Включить/отключить
- Настроить отдельно
- Разрабатывать независимо

### Доступные модули:
- **Auth** - Система авторизации и ролей
- **Logging** - Логирование действий пользователей
- **Admin** - Административные команды
- **NetBox** - Интеграция с NetBox (отключен)

## 🛠️ Разработка

### Добавление нового модуля:

1. Создайте файл в `app/handlers/your_module.py`
2. Добавьте роутер в `app/main.py`
3. При необходимости добавьте конфигурацию в `config.py`

### Структура обработчика:
```python
from aiogram import Router
from aiogram.types import Message
from aiogram.filters import Command

router = Router()

@router.message(Command("your_command"))
async def your_handler(message: Message, db_user, user_role):
    await message.answer("Ваш ответ")
```
