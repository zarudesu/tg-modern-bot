## 🔗 Готовность к интеграции

### **n8n Webhook структура подготовлена:**
```json
{
 "source": "telegram_bot",
 "event_type": "work_journal_entry",
 "data": {
   "work_entry": {
     "date": "2025-08-01",
     "company": "Ива",
     "duration": "45 мин", 
     "description": "Описание работ",
     "is_travel": false,
     "workers": ["Тимофей", "Дима"],
     "workers_count": 2
   },
   "creator": {
     "name": "Kostya",
     "telegram_id": 28795547
   }
 }
}
```

### **Переменные окружения готовы:**
```bash
N8N_WEBHOOK_URL=https://your-n8n.com/webhook/work-journal
GOOGLE_SHEETS_ID=your_google_sheets_id_here
```
